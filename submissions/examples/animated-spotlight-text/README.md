# Animated Spotlight Text

## What it does

Text revealed by a moving spotlight gradient that sweeps across the letters like a cinematic light.

## How to use it

<h2 class="ease-spotlight-text">CINEMATIC TEXT</h2>

## Variants

- ease-spotlight-fast - 1s sweep
- ease-spotlight-slow - 5s sweep
- ease-spotlight-glow - Neon glow colors
- ease-spotlight-blue - Cyan/blue theme
- ease-spotlight-gold - Gold/yellow theme
- ease-spotlight-word - Per-word animation

## Features

- Moving gradient spotlight
- Multiple color themes
- Speed control
- Per-word staggered animation
- Pause on hover

## Why it fits EaseMotion CSS

- Pure CSS gradient animation
- Cinematic aesthetic
- Staggered word option
- Smooth infinite sweep
