# Animated Stack of Photos

## What it does

A 3D stack of photos that fans out, spreads apart, or raises up on hover.

## How to use it

<div class="ease-photo-stack">
    <div class="ease-photo">Photo 1</div>
    <div class="ease-photo">Photo 2</div>
    <div class="ease-photo">Photo 3</div>
</div>

## Variants

- ease-stack-fan - Photos fan out like a deck
- ease-stack-spread - Photos spread horizontally
- ease-stack-raised - Photos lift upward

## Features

- 3D rotation and translation
- Elastic easing on hover
- Multiple stack animations
- Polaroid-style shadows

## Why it fits EaseMotion CSS

- Pure CSS 3D transforms
- Multiple animation styles
- Photo gallery aesthetic
- Smooth elastic easing
