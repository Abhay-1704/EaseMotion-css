# Animated Word Scrambler

## What it does

Text that scrambles randomly on hover, then reveals the original word with a cyberpunk effect.

## How to use it

<div class="ease-word-scrambler" data-text="TARGET TEXT">Initial Text</div>

## Features

- Random character scrambling
- Progressive reveal animation
- Hover-triggered scrambling
- Speed control (fast/slow)
- Gradient text styling

## Variants

- Hover to scramble
- Scramble All button
- Fast/Slow speed modes
- Reset functionality

## Why it fits EaseMotion CSS

- Pure CSS + JavaScript animation
- Cyberpunk aesthetic
- Interactive text effect
- Smooth progressive reveal
